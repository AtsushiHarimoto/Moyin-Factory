# Moyin Dev Dashboard — Developer Workflow Manager

## Purpose

Full-featured dashboard for managing AI coding agent configurations, skill profiles, and development session analytics across Claude Code, Codex, and other AI tools.

## Capabilities

- **Skills Management**: Install, verify, and sync AI agent skills across platforms
- **Profile Switching**: Save/load skill presets for different development workflows
- **Cross-Platform Sync**: Configuration synchronization between macOS and Windows
- **Session Analytics**: Track conversation history and productivity insights
- **Knowledge Base**: Built-in wiki for documentation management
- **Email Reports**: Automated analysis report delivery

## Architecture

- **CLI Tool**: Commander.js-based command-line interface
- **Web Dashboard**: React 18 + TypeScript + Vite
- **Backend**: Express + SQLite + Nodemailer
- **Data Visualization**: Recharts + Mermaid diagrams

## Open-Source Component

Available as [Moyin-Dev-Dashboard](https://github.com/AtsushiHarimoto/Moyin-Dev-Dashboard).
