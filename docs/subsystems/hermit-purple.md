# Hermit Purple — AI Trend Research Tool

## Purpose

Automated research and analysis tool that monitors AI industry trends across multiple sources, generating synthesized reports with sentiment analysis.

## Capabilities

- **Multi-Source Crawling**: GitHub, Reddit, YouTube, Bilibili, and web search
- **Plugin Architecture**: Extensible analysis pipelines (AI Trends, Social Radar, AI Business)
- **LLM-Powered Synthesis**: Gemini/Ollama-driven report generation
- **Keyword Discovery**: Automatic trend keyword identification
- **MCP Integration**: Exposes tools via Model Context Protocol

## Technology

- Python (asyncio)
- SQLite for local storage
- Gemini API / Ollama for LLM
- Plugin-based pipeline architecture

## Open-Source Component

Available as [hermit-purple](https://github.com/AtsushiHarimoto/hermit-purple).
