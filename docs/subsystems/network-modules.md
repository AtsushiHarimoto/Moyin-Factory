# Moyin Network Modules — HTTP Client Library

## Purpose

Shared TypeScript HTTP client library providing request management with deduplication, retry logic, tracing, and i18n support.

## Features

- **Request Manager**: Centralized HTTP request handling with automatic deduplication
- **Retry Logic**: Configurable retry strategies with exponential backoff
- **Request Tracing**: Built-in tracing for debugging and monitoring
- **Mock Transport**: Testing utilities for simulating network responses
- **Pinia Integration**: State management store for network state
- **i18n Support**: Internationalized error messages

## Technology

- TypeScript
- Vitest for testing
- Pinia store integration

## Open-Source Component

Available as [Moyin-Network-modules](https://github.com/AtsushiHarimoto/Moyin-Network-modules).
